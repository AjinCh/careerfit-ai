# CareerFit AI

CareerFit AI is an AI-powered job market intelligence and skill gap recommendation platform for international graduates and early-career professionals applying for Data Science, Machine Learning, Applied AI, Data Analyst, Data Engineering, MLOps, and GenAI roles in Germany.

## Project Purpose

Many international graduates and early-career professionals struggle to understand what skills employers expect, how their CV compares to job descriptions, and what they should learn next.

CareerFit AI helps candidates:

- Understand skill requirements from job descriptions
- Compare their current profile against role expectations
- Identify skill gaps
- Receive structured learning recommendations
- Improve their CV for clarity, ATS readability, and relevance

## Important Positioning

CareerFit AI is a candidate self-improvement and career guidance tool.

It is not:

- An employer-side screening system
- A candidate rejection tool
- A hiring decision system
- A tool for ranking people unfairly

The platform is designed to be ethical, explainable, GDPR-aware, and beginner-friendly.

## Target Users

Primary users:

- International graduates in Germany
- Early-career data professionals
- Students transitioning into data and AI roles
- Career switchers with basic technical skills

Target roles:

- Data Analyst
- Data Scientist
- Machine Learning Engineer
- Applied AI Engineer
- Data Engineer
- MLOps Engineer
- GenAI Engineer

## MVP Features

The first MVP will include:

1. Job description input
2. Basic skill extraction using a predefined skill taxonomy
3. CV/profile skill input
4. Skill gap comparison
5. Learning recommendations
6. ATS-friendliness checklist
7. Simple dashboard interface
